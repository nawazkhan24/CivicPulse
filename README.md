CivicPulse

Stay informed about what your city government is deciding — automatically.

 What it does
CivicPulse monitors your local government's meeting agendas and sends you
an email alert whenever topics you care about are discussed.

 Features
- User registration and login
- Custom keyword tracking
- Automatic daily agenda scraping
- Instant email notifications

 Tech Stack
- Node.js + Express
- MongoDB
- Nodemailer
- Node-cron
- Cheerio + Axios

 Setup
1. Clone the repo
2. Run `npm install`
3. Create `.env` file with your credentials
4. Run `node server.js`
